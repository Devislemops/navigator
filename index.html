Basic Commands

  • ls – list directory contents.
    Example: ls -l /home
  • cd – change directory.
    Example: cd /var/log
  • pwd – print working directory.
    Shows the full path of the current directory.
  • tree – displays directories and files in tree format.
    Example: tree /etc
  • touch – create empty files or update timestamps.
    Example: touch newfile.txt
  • mkdir – create directories.
    Example: mkdir /tmp/testdir
  • rm – remove files or directories.
    Example: rm oldfile.txt
  • cp – copy files or directories.
    Example: cp file1.txt /tmp/
  • mv – move or rename files/directories.
    Example: mv file1.txt file2.txt
  • cat – display file content.
    Example: cat /etc/hostname
  • less / more – view file content page by page.
    Example: less /var/log/syslog
  • head / tail – view first/last lines of a file.
    Example: head -n 20 /var/log/syslog
  • grep – search text patterns.
    Example: grep "error" /var/log/syslog
  • sed – stream editor for text manipulation.
    Example: sed 's/old/new/g' file.txt
  • awk – text processing & reporting.
    Example: awk '{print $1}' file.txt
  • sort – sort lines in text files.
    Example: sort file.txt
  • cut – extract columns from text.
    Example: cut -d ":" -f1 /etc/passwd
  • wc – word, line, byte count.
    Example: wc -l file.txt

Files & Archives

  • tar – archive files.
    tar -czvf backup.tar.gz /home/user
  • gzip / gunzip – compress / decompress files.
    gzip file.txt
  • zip / unzip – zip archives.
    zip archive.zip file1 file2
  • find – search files in directories.
    find /var/log -name "*.log"
  • locate – fast file search using database.
    locate nginx.conf
  • du – disk usage.
    du -sh /home/user
  • df – disk free space.
    df -h

Processes & Resources

  • top / htop – live view of processes, CPU, memory usage.
  • ps / pstree – list processes in different formats.
  • kill / pkill – terminate processes by PID or name.
  • nice / renice – set or change process priority.
  • free / vmstat / iostat – monitor memory and I/O.

Users & Permissions

  • useradd / usermod / passwd – add/modify users.
  • groupadd / groupmod – manage groups.
  • chmod / chown / chgrp – permissions & ownership.
  • sudo / visudo – manage admin privileges.
  • umask – default permissions for new files.

Network

  • ip / ss / netstat – view network interfaces, sockets, connections.
  • nmcli – manage network connections.
  • ping / traceroute / mtr – test connectivity.
  • dig / host / nslookup – DNS lookup.
  • curl / wget – fetch URLs and files.
  • scp / rsync – remote copy/sync files.
  • nmap – network scanning.

System & Services

  • systemctl / service – start/stop/manage services.
  • journalctl – view system logs.
  • dmesg – kernel messages.
  • reboot / shutdown – restart or power off system.

Storage & LVM

  • lsblk – list disks and partitions.
  • mount / umount – mount or unmount filesystem.
  • parted / fdisk – disk partitioning.
  • mkfs – create filesystem.
  • fsck – check filesystem.
  • LVM: pvcreate, vgcreate, lvcreate, lvextend, lvs/vgs/pvs

Security

  • fail2ban-client – monitor and ban suspicious IPs.
  • ssh-keygen – generate SSH keys.
  • chage – password expiry management.
  • ufw / firewall-cmd – firewall management.

Logs & Troubleshooting

  • journalctl – view system logs.
  • tail -f – follow log files in real-time.
  • dmesg – kernel messages.
  • lsof – list open files.
  • strace – trace system calls.
  • tcpdump – capture network traffic.

DevOps

  • git – add, commit, push, branch, revert, stash.
  • docker – build, run, exec, logs, ps, stats.
  • kubectl – manage pods, deployments, nodes.
  • ansible – ad-hoc commands, playbooks.
  • terraform – init, plan, apply, destroy.

HPC Commands

  • squeue – view jobs in Slurm queue.
  • scontrol – manage Slurm jobs.
  • sbatch – submit batch jobs.
  • scancel – cancel jobs.
  • sinfo – view cluster info.
  • sacct – view accounting of jobs.
  • slurmd status check – check node daemons.
  • nfsstat / showmount – NFS monitoring.
  • ibstat / ibv_devinfo – InfiniBand status.
